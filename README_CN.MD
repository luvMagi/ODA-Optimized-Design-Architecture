## 🌐 项目理念

**ODA（Optimized Design Architecture）** 是一套面向现代系统开发的设计自动化平台。通过统一定义、模块协同与模板驱动机制，ODA 实现了从“设计一次”出发，自动生成 **Web画面设计书、报表定义、数据库建模文档** 等各类开发所需交付物。

> **Design once, generate all.**  
> 一次定义，多点输出。最大限度提升设计一致性、开发效率与交付质量。

---

## 🧩 模块体系

ODA 平台按设计目标与输出类型，划分为三大功能模块：

### 1. 画面设计辅助模块（UI Design Support）

- 支持画面功能信息、控件定义、事件流程等的标准化建模  
- 自动生成符合规范的画面详细设计书（Excel 等）

### 2. 报表定义辅助模块（Report Design Support）

- 支持输出样式、CSV 项目、控制信息的统一定义  
- 自动生成报表模板、SVF结构配置、CSV测试数据等

### 3. 数据库建模辅助模块（Database Design Support）

- 支持从ER逻辑到DDL转换，字段映射与差异分析  
- 自动输出数据库结构定义文档、过程依赖图、表设计一览表等

---

## 📁 应用场景

- 💻 Web系统画面详细设计书 批量生成  
- 📄 报表输出定义（含CSV/SVF/控制信息）一键生成  
- 🗃 数据库建模文档（ER结构/字段映射/差异分析）统一管理  
- 🧠 支持AI辅助设计框架开发，便于集成 Copilot、自动化生成器  
- 📦 项目模板可扩展，适配各类设计规范和格式需求（Excel / Markdown / XML等）
